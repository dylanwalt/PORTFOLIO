Smart Pot Plant

A fully customized Smart Pot Plant system that monitors soil moisture, temperature, and humidity. The project uses an ESP32 microcontroller and sensors to track environmental data and sends alerts when the plant needs watering. This project combines hardware engineering, sensor integration, and real-time data analysis.
